Projet PFA - application de Cybersécurité d'un Applications Bancaires

Introduction au Projet

L’entreprise fictive propose un service de banque en ligne via l’application web Aabil, hébergée sur une machine virtuelle (VM) avec une base de données contenant des informations sensibles. Une équipe d’administration gère les accès et les services.

Les objectifs principaux sont :

  • Développer une application web bancaire avec une base de données
  • Maîtriser les outils de pentesting pour identifier les vulnérabilités
  • Attaquer l’application web pour obtenir un accès non autorisé aux données sensibles à l’aide de Scrapy et Netcat.
  • Sécuriser le réseau et les services contre ces attaques en utilisant Suricata, Splunk et OpenVAS.
